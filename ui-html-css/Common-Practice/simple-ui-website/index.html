HTML là gì ?

Là một ngôn ngữ đánh dấu dùng để mô tả nội dung tài liệu web (trang web)

Nguyên tắc dựng cấu trúc các thẻ HTML

  • Các thẻ H1 - H6 và P không thể bọc lẫn nhau.
  • Thẻ P chỉ có thể bọc phần tử dạng inline
  • Phần tử, thẻ cấp độ Block level có thể bọc inline level.
  • Thẻ inline chỉ có thể bọc inline và không nên dùng đơn lẻ.
  • Sử dụng Div để nhóm các phần tử liên quan với nhau.
