Mỹ nhân Hàn tranh thủ khoe eo thon khi đi xem thời trang

Hôm 16/10, sao nữ Hàn kéo nhau đến Tuần lễ thời trang Seoul với phong cách dễ nhận ra nhất là kheo eo thon. Tuy nhiên, không phải ai cũng ghi điểm.

Ngây ngất vì mỹ nhân Hoa Ngữ quá đẹp trên tạp chí tháng 10. 8 mỹ nhân Việt mặc xấu trên thảm đỏ mở màn Tuần thời trang
